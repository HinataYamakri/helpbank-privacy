データの削除方法

お手伝いバンクでは、アカウントを削除することなく、個別のデータを削除できます。プライバシーを保護しながら、必要なデータだけを残すことができます。

📱 アプリ内で削除できるデータ

データの種類 削除方法 削除の影響
お手伝い（個別） お手伝い管理画面 → 削除ボタン 該当のお手伝いとポイント履歴が削除
ご褒美（個別） ご褒美管理画面 → 削除ボタン 該当のご褒美と交換履歴が削除
子どものアカウント 設定 → こどもの管理 → 削除ボタン その子どものデータすべて削除
完了済みのお手伝い 承認済みタブ → アーカイブ/削除 履歴から削除（ポイントは維持）

🗑️ お手伝いを削除する

保護者モードでの操作

  1. 保護者モードにログイン
  2. 「お手伝い管理」タブを選択
  3. 削除したいお手伝いの「編集」ボタンをタップ
  4. 「削除」ボタンをタップ
  5. 確認ダイアログで「削除する」を選択

🎁 ご褒美を削除する

保護者モードでの操作

  1. 保護者モードにログイン
  2. 「ごほうび管理」タブを選択
  3. 削除したいご褒美の「編集」ボタンをタップ
  4. 「削除」ボタンをタップ
  5. 確認ダイアログで「削除する」を選択

👤 子どものアカウントを削除する

特定の子どものデータのみ削除

  1. 保護者モードにログイン
  2. 「設定」タブを選択
  3. 「こどもの管理」セクションを表示
  4. 削除したい子どもの横にあるゴミ箱アイコンをタップ
  5. 確認ダイアログで「削除する」を選択
⚠️ 注意：子どものアカウントを削除すると、その子どもの以下のデータがすべて削除されます：
  • 獲得ポイント
  • お手伝い履歴
  • ご褒美交換履歴

📊 履歴データをクリアする

古いデータの整理

  1. 保護者モードにログイン
  2. 「承認」タブで完了済みのお手伝いを確認
  3. 不要な履歴を選択
  4. 「アーカイブ」または「削除」を選択

アーカイブ：進捗レポートには残るが、一覧からは非表示

削除：完全に削除（ポイントは維持される）

💡 削除のヒント

  • 削除したデータは復元できません
  • ポイント残高は、履歴を削除しても維持されます
  • 新しいお手伝いやご褒美はいつでも追加可能です
  • 子どものアカウントは再作成できます
